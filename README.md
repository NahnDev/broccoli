### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/NahnDev/broccoli.git
   cd broccoli
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

### Running the Development Server

```sh
npm run dev
```
